# GitHub Sentinel v0.0.3

## 项目简介

GitHub Sentinel 是一款面向开发者和项目管理人员的开源工具型 AI Agent，用于订阅 GitHub/Gitee 仓库、获取仓库动态、整理项目变化并生成 Markdown 简报，帮助团队把分散的提交、Issue 和 Pull Request 信息转化为可阅读、可追踪的项目进展材料。

## 此版本支持的功能特性

- 新增 Gradio 可视化界面入口，支持在页面中创建订阅、查看订阅列表、通过操作列删除订阅，并在报告中心完成报告生成与 Markdown 正文预览。
- 报告中心支持选择订阅仓库后主动查询报告列表，也支持通过“查询”按钮按最近时间范围或自定义日期范围手动刷新报告列表。
- 报告中心将“报告列表查询日期”和“生成报告日期”拆分为两组独立日期条件，默认填入最近一周，便于分别控制查看范围和生成范围。
- 日期选择器完成中文化与弹层位置优化，月份、星期和操作按钮使用中文展示，弹层在页面内完整显示且不会触发页面级布局错乱。
- 报告预览区域调整为模块内部滚动，避免报告正文过长时带来页面整体滚动。
- 报告数据增加日期周期记录，报告名称按仓库和日期范围生成；同一订阅仓库、同一日期范围重复生成时更新既有报告，减少重复报告堆积。
- 仓库事件汇总范围扩展到 Pull Request 事件，报告整理能力从 Push、Issue 扩展为 Push、Issue、Pull Request。
