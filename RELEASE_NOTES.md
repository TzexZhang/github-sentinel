# GitHub Sentinel v0.0.4

## 项目简介

GitHub Sentinel 是一款面向开发者和项目管理人员的开源工具型 AI Agent，用于订阅 GitHub/Gitee 仓库、获取仓库动态、整理项目变化、生成项目简报，并通过配置好的通知通道把项目进展同步给订阅者，帮助团队把分散的提交、Issue 和 Pull Request 信息转化为可阅读、可追踪、可分发的项目进展材料。

## 此版本支持的功能特性

- 支持单用户、多仓库订阅管理，可在 Dashboard 中创建、查看、修改和删除仓库订阅。
- 支持按订阅维度配置抓取间隔，系统会在订阅到期后获取当前周期内的新事件并生成报告。
- 支持订阅与通知通道绑定，通知类型覆盖 SMTP 邮箱、企业微信自建应用消息和通用 Webhook。
- 支持在创建或修改订阅时选择通知类型并填写通知目标；企业微信通知目标使用成员账号/UserID，SMTP 通知目标使用收件人邮箱地址。
- 支持报告生成后创建通知任务，由应用内 Notification Worker 统一投递通知并记录发送状态、失败原因和重试信息。
- 支持手动生成报告时选择是否发送通知，勾选后会把生成的报告加入通知投递流程。
- 支持将报告 Markdown 转换为适合不同通道展示的内容，避免邮箱、企业微信和 Webhook 直接收到原始 Markdown 文本。
- 企业微信通知支持通过自建应用向成员账号推送文本消息，并预留接收消息服务器 URL、Token 和 EncodingAESKey 配置入口，便于后续部署到服务器后继续接入 API 回调。
